# Privacy Policy

**BotC Scripts Bot** ("the Bot") is a Reddit app built with Devvit that automatically links Blood on the Clocktower scripts from [botcscripts.com](https://botcscripts.com).

## Data We Collect

The Bot does not collect, store, or retain any personal data.

When a comment containing `[[script name]]` is detected, the Bot:
1. Reads the comment text provided by Reddit's platform.
2. Sends only the script name (extracted from the comment) to the public API at `botcscripts.com` as a search query.
3. Posts a reply comment on Reddit.

No usernames, user IDs, comment IDs, or any other information are stored or transmitted beyond what is described above.

## External Services

The Bot makes requests to `botcscripts.com` solely to search for scripts by name. Please refer to the [botcscripts.com](https://botcscripts.com) privacy policy for information about how that service handles requests.

## Reddit Platform

The Bot operates within Reddit's platform and is subject to [Reddit's Privacy Policy](https://www.reddit.com/policies/privacy-policy).

## Changes

This policy may be updated at any time. Any changes will be reflected in this document.

## Contact

For questions or concerns, contact the developer via the [botcscriptbot app page](https://developers.reddit.com/apps/botcscriptbot).
